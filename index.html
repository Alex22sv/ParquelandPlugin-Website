Parqueland

Parqueland es un plugin de Minecraft para el servidor de roleplay "Parqueland" en español pero también puede usarse en otros servidores.

  • GitHub
  • SpigotMC

Funciones

¿Qué funciones tiene este plugin? ¡Las más necesitadas en tu servidor roleplay!

Menú de admins personalizado

Con el comando /adminmenu podrás acceder a una interfaz que te permitirá moderar jugadores fácilmente.

Mensajes de bienvenida y despedida personalizados

En el archivo config.yml podrás habilitar los permisos personalizados enviados cuando jugadores entren al servidor o salgan de él. Compatible con formatos de colores.

Tus propias reglas

En el archivo config.yml podrás escribir tus propias reglas del servidor que se mostrarán cuando un jugador ejecute /reglas.

Roleplay con DIOSES

Con el comando /dioses podrás enviar mensajes como si fueses parte del grupo de "DIOSES".

Comandos

Aquí encontrarás la lista de comandos con sus respectivos permisos requeridos.

Comando Descripción Permisos
/adminmenu Abre el menú de admins para banear/expulsar/mutear/congelar jugadores. parqueland.AdminMenuCommand
/alimentar <jugador> Quita el hambre a un jugador. parqueland.Alimentar
/curar <jugador> Cura a un jugador. parqueland.CurarCommand
/dioses <jugador> <mensajes> Envía mensajes como autor 'DIOSES'. parqueland.DiosesCommand
/enviarobjeto <jugador> <cantidad> Envía una cantidad de objetos que tengas en la mano a otro jugador. parqueland.EnviarObjetoCommand
/rayo <jugador> Lanza un rayo a un jugador. parqueland.RayoCommand
/reglas Obtén las reglas del servidor. Ninguno
